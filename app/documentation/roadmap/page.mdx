# Roadmap

The reference implementation plan sequences infrastructure, AI pipeline work, frontend review experiences, marketplace plumbing, and listing readiness.

| Phase | Duration | Focus |
| --- | --- | --- |
| Infrastructure | Weeks 1–2 | IaC for storage, identity, and supporting services |
| AI pipeline | Weeks 3–5 | Textract + Bedrock (Converse API) integration |
| Frontend | Weeks 6–8 | Review UI and document explorer |
| Marketplace | Weeks 9–10 | ResolveCustomer + BatchMeterUsage flows |
| Listing | Weeks 11–12 | Security review and go-live |

## Extensibility

Future depth can add richer RBAC, custom model routes (post-MVP), and deeper analytics—while keeping Postgres authoritative and OpenSearch rebuildable.
